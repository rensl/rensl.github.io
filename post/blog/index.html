#报警接口web服务

2019.12.18 修改历史报警请求字段（is_event, status, is_confirm, confirm_user）

##实时报警

1 说明

本文件定义了MetroView4.0实时报警web接口

2 接口定义

2.1 获取全部报警

（1）接口格式

url:/alarm/real/pull
method:post
data:
{
	"type":"record",
    "fields":[
    	{"name":"userid","type":"int32"},
        {"name":"version","type":"int32"},
    ]
}

（2）参数说明

参数名称 参数说明 是否必选 备注
userid 用户id 是
version 版本号 是 来自服务端上次回复的version

（3）返回值说明

data:
{
	"type":"record",
    "fields":[
    	{"name":"version","type":"int32"},
        {
        	"type":"array",
            "name":"alarmlist",
            "items":{
            	"type":"record",
                "fields":[
                	{"name":"alarmid","type":"string"},
                    {"name":"is_event","type":"int32"},
                    {"name":"type","type":"int32"},
                    {"name":"level","type":"int32"},
                    {"name":"valuetype","type":"int32"},
                    {"name":"dival","type":"int32"},
                    {"name":"aival","type":"double"},
                    {"name":"repeat","type":"int32"},
                    {"name":"status","type":"int32"},
                    {"name":"confirm","type":"int32"},
                    {"name":"suppress","type":"int32"},
                    {"name":"repair","type":"int32"},
                    {"name":"print","type":"int32"},
                    {"name":"pushpic","type":"int64"},
                    {"name":"picname","type":"string"},
                    {"name":"create_time","type":"int64"},
                    {"name":"remove_time","type":"int64"},
                    {"name":"confirm_time","type":"int64"},
                    {"name":"suppress_time","type":"int64"},
                    {"name":"confirm_user","type":"string"},
                    {"name":"suppress_user","type":"string"},
                    {"name":"station","type":"string"},
                    {"name":"subsystem","type":"string"},
                    {"name":"devicetype","type":"string"},
                    {"name":"deviceid","type":"string"},
                    {"name":"point","type":"string"},
                    {"name":"message","type":"string"}
                ]
            }
        }
    ]
}
名称 说明 备注
alarmid 报警id tag.type.value
is_event 是否事件报警
type 报警类型
level 报警等级
valuetype 值类型 0：di，1：ai
dival
aival
repeat 重复产生次数
status 是否消除 0：已消除，1：未消除
confirm 是否确认 0：未确认，1：已确认
suppress 是否抑制 0：未抑制，1：已抑制
repair 挂牌状态
print
pushpic
picname
create_time 产生时间 毫秒
remove_time 消除时间 毫秒
confirm_time 确认时间 毫秒
suppress_time 抑制/取消抑制时间 毫秒
confirm_user 确认用户
suppress_user 抑制/取消抑制用户
station
subsystem
devicetype
deviceid
point 点名 如"HZBJ”
message 报警描述

2.2 报警确认

（1）接口格式

url:/alarm/real/confirm
method:post
data:
{
	"type":"record",
    "fields":[
    	{"name":"userid","type":"int32"},
        {
        	"type":"array",
            "name":"alarmid",
            items:{
            	"type":"string",
            }
        }
    ]
}

（2）参数说明

参数名称 参数说明 是否必选 备注
userid 用户id 是
alarmid 报警id数组 是

（3）返回值说明

data:
{
	"type":"record",
    "fields":[
    	"name":"result",type:"string"
    ]
}
名称 说明 备注
result 结果 固定为“ok”

（4）示例

url:http://172.20.30.68:8688/alarm/real/confirm
method:post
data:
{
	"userid":65537,
    "alarmid":[
    	"Z63_HK_OPER_0_KZQXFK.15.0",
        "Z63_RVS_DLJ1_0_SDZTFK.15.0"
    ]
}
return:
{
	"result":"ok"
}

##历史报警

1 说明

本文件定义了MetroView4.0历史报警web接口

2 接口定义

2.1 历史报警查询

（1）接口格式

url:/alarm/his/query
method:post
data:
{
	"type":"record",
    "fields":[
    	{"name":"beginrow","type":"int32"},
        {"name":"needrow","type":"int32"},
        {"name":"is_event","type":"int32"},
        {"name":"type","type":"string"},
        {
        	"type":"array",
            "name":"level"
            "items":{
            	"type":"int32"
            }
        },
        {"name":"status","type":"int32"},
        {"name":"is_confirm","type":"int32"},
        {"name":"createtimebegin","type":"int64"},
        {"name":"createtimeend","type":"int64"},
        {"name":"station","type":"string"},
        {"name":"subsystem","type":"string"},
        {"name":"devicetype","type":"string"},
        {"name":"device","type":"string"},
        {"name":"confirm_user","type":"string"},
        {"name":"detailed","type":"int32"},
        {
        	"type":"array",
            "name":"stationlist",
            "items":{
            	"type":"string"
            }
        },
        {
        	"type":"array",
            "name":"subsystemlist",
            "items":{
            	"type":"string"
            }
        },
        {
        	"type":"array",
            "name":"devicetypelist",
            "items":{
            	"type":"string"
            }
        }，
        {
        	"type":"array",
            "name":"devicelist",
            "items":{
            	"type":"string"
            }
        }
    ]
}

（2）参数说明

参数名称 参数说明 是否必选 备注
beginrow 是 填-1返回以needrow行数为一页的最后一页
needrow 是
is_event 否
type 否
level 否 数组
status 否
is_confirm 否
createtimebegin 否 单位：秒
createtimeend 否 单位：秒
station 否
subsystem 否
devicetype 否
device 否
confirm_user 确认用户 否 不填则表示任意用户
detailed 否 1时取station,subsystem,devicetype和device,0时取stationlist,subsystemlist,devicetypelist，devicelist进行查询
stationlist 否
subsystemlist 否
devicetypelist 否
devicelist 否

（3）返回值说明

成功：
{
	"type":"record",
    "fields":[
    	{"name":"version","type":"int32"},
        {
        	"type":"array",
            "name":"alarmlist",
            "items":{
            	"type":"record",
                "fields":[
                	{"name":"alarmid","type":"string"},
                    {"name":"is_event","type":"int32"},
                    {"name":"type","type":"int32"},
                    {"name":"level","type":"int32"},
                    {"name":"repeat","type":"int32"},
                    {"name":"status","type":"int32"},
                    {"name":"confirm","type":"int32"},
                    {"name":"suppress","type":"int32"},
                    {"name":"create_time","type":"int64"},
                    {"name":"remove_time","type":"int64"},
                    {"name":"confirm_time","type":"int64"},
                    {"name":"suppress_time","type":"int64"},
                    {"name":"confirm_user","type":"string"},
                    {"name":"suppress_user","type":"string"},
                    {"name":"station","type":"string"},
                    {"name":"subsystem","type":"string"},
                    {"name":"devicetype","type":"string"},
                    {"name":"deviceid","type":"string"},
                    {"name":"point","type":"string"},
                    {"name":"message","type":"string"},
                    {"name":"action","type":"string"}
                ]
            }
        }
    ]
}
失败：
{
	"type":"record",
    "fields":[
    	{"name":"total","type":"int32"},
        {"name":"error","type":"string"}
    ]
}
